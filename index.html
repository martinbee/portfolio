Skeletons

Skeletons is an anonymous secret sharing board and my first distrubted web app. It was built with a Ruby on Rails with Postgresql back-end API and a Rails front-end, using javascript to speed up load times.

Tools: Ruby, Ruby on Rails, JavaScript, jQuery, Bootstrap, CSS, HTML, Heroku, Postgresql, and GIMP for design.

Bitter

Bitter is a twitter clone focused entirely upon allowing users to complain and share their 'FML' moments. Users can "Bitch" about their days and other users can "Commiserate" with them. Bitter is a full-stack Rails app and includes a fully fleshed follow system and javascript forms.

Tools: Ruby, Ruby on Rails, JavaScript, jQuery, Bootstrap, CSS, HTML, Heroku, and Postgresql.

Pineterest

Pineterest is a Pinterest clone for pine trees only. It was designed to replicate an art gallery experience and to let users share their pine tree photos. It was built with a Ruby on Rails with Postgresql back-end API and a Rails front-end, using javascript to create a seamless user experience.

Tools: Ruby, Ruby on Rails, JavaScript, jQuery, Bootstrap, CSS, HTML, Heroku, Postgresql, and GIMP for design.

Houston.Vote

Houston.Vote is phase one of a campaign to raise voting awareness across Houston. It was made to serve as a hub for voting resources with the goal of increasing voter turnout for local elections, by me and a team of Houstonians at the Houston Hackathon. It was built with a PHP back-end, a Wordpress front-end, and then given a custom design.

Tools: PHP, Wordpress, Photoshop, JavaScript, jQuery, CSS, HTML, and MySQL
